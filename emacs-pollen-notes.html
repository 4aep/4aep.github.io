Here are some notes and tips on using pollen with Emacs

Packages to use

  • Pollen-mode: Namely for the fact that it can easily insert lozenge, syntax highlighting, and start the Pollen server.

  • Magit: Fast and featureful git front-end. See also git gutter and forge

  • Yas-snippet: For easy insertion of more complex commands like links and also for a template/ draft / outline of your text.

  • Spellcheck: I use recomplete since it is fast and I don't need the redline.

  • Smartparens: Due to working with brackets, nesting is easy to deal with and automatically jumps you inside the brackets when you make it.

  • Consult: consult-line and consult-ripgrep are quite useful functions that make finding things way easier

  • Avy: It can do a lot of useful stuff, I mainly using it for yanking and googling things. It basically replaces fine mouse movements.

  • Neotree: Usually don't use it but it is nice to have a side overview of the whole project.

  • read-aloud.el: Not the cleanest, but it is useful for proof-reading.
